# Nashville Hospitality Target List

Checked on **May 12, 2026**.

## Pay Estimate Assumptions
- Tennessee has **no state income tax**.
- `Est. monthly take-home` is a **rough** estimate using about **80% of gross pay**.
- For `full-time` jobs without stated hours, I used **173 hours/month**.
- For `30 hours/week`, I used **130 hours/month**.
- For `25 hours/week`, I used **108 hours/month**.
- For `weekly pay` listings, I used **4.33 weeks/month**.
- For tipped jobs that only disclose base pay, the estimate is **base-only** unless the posting explicitly gave a combined hourly range.
