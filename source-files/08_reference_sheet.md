This file stores Matt’s approved professional references for use in:

## Reference handling rules
- Use references only when needed.
- Do not include them in cold outreach or standard cover letters.
- Do not volunteer phone numbers or emails unless the process calls for it.
- When possible, ask which references would be most relevant to the role before sending them.
- Match references to the lane and likely concerns.
