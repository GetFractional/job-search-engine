# Codex Windows Migration Bundle

This folder contains a GitHub-safe, sanitized copy of the Mac Codex migration bundle for setting up Codex on Windows.

Use `codex-mac-setup-export/MIGRATION_NOTES.md` first. The checked-in `config.toml` is intentionally redacted. Recreate credentials, OAuth connectors, MCP env values, browser login state, and machine-specific paths directly on Windows.

The local Desktop export on the Mac may contain the original `config.toml`; do not commit or paste that raw file.

The zip beside this README is the same sanitized content for easy download on Windows.
