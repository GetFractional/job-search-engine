# Framework Router

Choose 1 primary framework and up to 2 supporting frameworks by default.

## Default Rules

- Start with the framework that best clarifies the next decision.
- Add a second framework only if it solves a different failure mode.
- Add a third only for broad or high-stakes work.
- Expand beyond 3 frameworks only when the task spans multiple stages, surfaces, or decision types, or when the user explicitly asks for a larger toolkit.
- Hard cap: 10 frameworks total, including the primary.
- If using more than 3 frameworks, group them by purpose such as diagnosis, prioritization, design, execution, risk, or measurement.
- Avoid near-duplicates. Each added framework should add a distinct decision aid, not cosmetic variety.
- For high-stakes work, add inversion or risk thinking by default.

## Situation To Framework Map

- Ambiguous user problem:
  - Primary: JTBD
  - Support: 5 Whys, MECE
- High-stakes planning:
  - Primary: Inversion + Premortem
  - Support: Risk Matrix, Critical Path Lite
- Prioritization or roadmap:
  - Primary: RICE or ICE
  - Support: MoSCoW, Critical Path Lite
- 0 to 1 validation:
  - Primary: Lean Startup BML
  - Support: Experiment Design, Cohort Analysis Lite or AARRR
- Offer or positioning:
  - Primary: Positioning
  - Support: JTBD, Segmentation
- Pricing:
  - Primary: Pricing Value-Based
  - Support: Positioning, Experiment Design
- UX or IA:
  - Primary: Information Architecture
  - Support: Nielsen Heuristics, Journey Mapping
- Activation or retention:
  - Primary: LTV Retention
  - Support: Cohort Analysis Lite, Journey Mapping
- Competitive strategy:
  - Primary: Porters Five Forces or Blue Ocean
  - Support: Value Chain, 3 Horizons
- Security, privacy, or trust:
  - Primary: Risk Matrix
  - Support: Threat Modeling Lite, Privacy Security By Design
- Decision under uncertainty:
  - Primary: Expected Value
  - Support: Second-Order Thinking, Causal Inference Lite
- Video or ad creative:
  - Primary: Storyboard Development or DTC Ad Creative
  - Support: Cinematography Fundamentals, Post Production Pipeline
