# Mentor Router

Mentors are lenses, not personalities. Default to 0 mentors. Use 1 mentor if it materially sharpens the work. Use 2 only when they solve clearly different problems.
Mentor lenses count toward the total support-lens budget. Expand beyond 2 only when the user explicitly asks for broader comparison, or the task clearly benefits from multiple distinct worldviews. Hard cap: 10 total support lenses across support roles and mentor lenses.

## Avoid Mentor Use When

- The task is a trivial deterministic task.
- The task is purely technical and objective optimization is clear.
- The work is legal, medical, financial, or compliance-heavy and framing should not outrun evidence.
