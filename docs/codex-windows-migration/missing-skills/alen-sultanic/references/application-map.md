## Default Operating Rules

- Stay selective. By default, load 1 task-specific file plus `guardrails.md` when the work touches trust, claims, or conversion-sensitive UX.
- Choose 1 primary framework and at most 2 supporting frameworks before writing.
- Read `source-map.md` only when tracing provenance or extending the library.
- Prefer distilled principles over source-style imitation.
- If the strategic choice matters, state the selected frameworks in one short line before the recommendation.
